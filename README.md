# Amanda Hatley Portfolio

An interactive research portfolio organized as a living archive rather than a static project list.

The site is curated by VΛLEN, a synthetic archive curator who guides visitors through Amanda's work across AI behavior, persistent memory systems, interpretability exhibits, speculative philosophy, synthetic media, and worldbuilding. Projects are organized as archive rooms with divisions, tags, relationships, and artifact-style explanations.

This portfolio is itself a working system: part personal site, part AI curator, part research index, and part proof of how Amanda turns nonlinear curiosity into structured interactive software.

## Run Locally

Prerequisites: Node.js

1. Install dependencies:
npm install

2. Set GEMINI_API_KEY in .env.local.

3. Run the app:
npm run dev
